Spring Boot 通过CORS解决跨域问题

· 2021-05-18 ·

跨域可以通过前端处理 也可以通过后端处理
同源策略 ： 协议 域名 端口都要相同
传统跨域解决方案 JSONP 只支持GET请求 ×
CORS （跨域源资源共享 Cross-Origin Resource Sharing）W3C标准

1.在controller上添加@CrossOrigin(origins="客户端域")

@RestController
public class HelloController {
    @GetMapping("/hello")
    @CrossOrigin(origins = "http://localhost:8081")
    public String hello(){
        return "hello cors";
    }
}

但这个缺点是得写好多次
2.WebMvcConfig类 实现 WebMvcConfiger类 并重写addCordMapping方法

@Configuration
public class WebMvcConfig implements WebMvcConfigurer {
    @Override
    public void addCorsMappings(CorsRegistry registry) {
        registry.addMapping("/**").allowedOrigins("http://localhost:8081")
                .allowedHeaders("*")//允许的请求头
                .allowedMethods("*")//允许的方法
                .maxAge(30*1000); //有效期
    }
}
